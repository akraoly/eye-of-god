# Vision — L'Œil de Dieu

## Concept

Un compagnon numérique personnel qui évolue sur 10 ans.
Pas un assistant générique. Un système conçu pour une seule personne, qui apprend d'elle, se souvient d'elle, et devient de plus en plus puissant avec le temps.

## Philosophie

1. **Évolution continue** — chaque conversation le rend plus intelligent
2. **Modularité totale** — chaque composant est remplaçable indépendamment
3. **Sécurité by design** — toute action système est whitelistée et loguée
4. **Centré utilisateur** — tout tourne autour d'une seule personne
5. **Préparation multimodale** — conçu pour accueillir voix, vision, capteurs

## Roadmap

### Phase 1 — MVP (actuel)
- Chat intelligent avec Claude
- Mémoire utilisateur automatique
- Agents de base (cyber, life, system)
- Interface web React

### Phase 2 — Intelligence
- Recherche vectorielle (ChromaDB/FAISS)
- Embeddings pour mémoire sémantique
- Résumé automatique des longues conversations
- Détection automatique de contexte

### Phase 3 — Agents avancés
- Agent cyber complet (Nmap, vulnérabilités, CTF)
- Agent productivité (calendrier, rappels, projets)
- Agent décisionnel (analyse de situations)
- Orchestration multi-agents

### Phase 4 — Multimodal
- Voix (Whisper → TTS)
- Vision (analyse d'images/screenshots)
- Intégrations externes (fichiers, web, APIs)

### Phase 5 — Autonomie
- Tâches planifiées
- Surveillance continue
- Auto-apprentissage structuré
